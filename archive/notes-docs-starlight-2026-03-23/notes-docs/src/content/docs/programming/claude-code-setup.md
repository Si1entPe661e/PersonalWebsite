# Claude Code Setup for Research Work

## What I want from an assistant

I do not want an assistant to replace judgment. I want it to reduce friction around repetitive structure, environment checks, and the translation from a vague task to a clean first draft.

## The useful constraints

For research-oriented coding work, three habits matter:

- keep prompts tied to a real file tree
- verify outputs instead of trusting confidence
- treat the assistant as a collaborator on process, not as an oracle on content

## Why this belongs on an academic site

Academic work increasingly includes code, data plumbing, document generation, and environment management. Workflow is no longer an aside. It is part of the craft.
