# A Reproducible Research Environment

## The simple rule

Every project should be easier to reopen than to remember. If reopening a folder requires memory, the system is too fragile.

## What helps

- predictable directory names
- a clear inputs and outputs split
- scripts that can be rerun without manual patching
- short notes on why a strange decision was made

## What this note is for

This page is a running place to collect the boring habits that keep empirical work from becoming archaeological work.
