# Linear Algebra for Economic Thinking

## Why this note exists

Much of the linear algebra used in economics is conceptually simple but easy to forget when it becomes buried inside notation. I keep this note as a reminder of what the objects are doing, not only how to manipulate them.

## Repeated themes

- basis changes and interpretation
- eigenvalues as summaries of dynamic behavior
- projection geometry in regression
- matrix decompositions as ways of clarifying structure

## The standard I want

If a matrix operation cannot be explained geometrically or behaviorally, I usually do not yet understand it well enough.
