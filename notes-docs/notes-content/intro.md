# Notes Library

This site is where I keep the slower, reusable part of my working knowledge. It is not a chronological blog. It is a small library built to support rereading, cross-reference, and practical use in research.

## What belongs here

- economics notes that clarify core frameworks and empirical strategies
- mathematics notes that keep useful tools close to hand
- programming notes about workflow, reproducibility, and research tooling

## How to use it

Treat the left sidebar as shelves, not as a feed. Some pages are compact reference notes. Others are working documents that will grow into longer sequences over time.
